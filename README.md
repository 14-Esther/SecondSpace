# SecondSpace 🌌

A modern marketplace web application built using **HTML, CSS, and JavaScript** with a vibrant tropical-inspired UI.

# Project Demo -

## ✨ Features

* Responsive design
* Animated floating cards
* Modern dashboard UI
* Search & filtering
* Smooth transitions
* Glassmorphism effects

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript

## 🚀 Run Locally
